最新版 v2rayN 下载📥，更新详情

更新时间：2024 年 05 月 08 日

1、介绍 #

v2rayN 是 Windows 系统下的代理软件客户端，功能强大且支持 VMess、VLESS、Trojan、Socks、Shadowsocks 等多种代理协议。

2、下载 #

⭐ 加速下载 | v2rayN 官网 Github 仓库地址为：https://github.com/2dust/v2rayN

如果从仓库下载，记得下载时要下载 v2rayn-core.zip 这个文件，这个文件包含了 V2Ray 核心文件，可以直接使用的发行版本，新手使用建议下载 稳定版本，即版本号后标记为 Latest 的版本。

3、版本更新内容 #

  • 添加出站 HTTP 支持（可以做为分组的前置代理）#4896
  • 添加了 tls 分片（Fragment）支持（只在Xray-core使用）#4900
  • 自定义配置前置Socks服务，将使用Xray或sing-box(Tun模式) #4855
  • 迁移geosite & geoip 为ruleset (sing-box v1.8.0+)
    1. 路由规则中的geosite & geoip会自动从Xray格式转换到ruleset格式
    2. 自定义DNS中的geosite & geoip会自动转换到ruleset格式；如果已经是ruleset则不会处理。默认DNS已经是ruleset
    3. 自动在路由中添加srs文件引用
  • 默认的路由规则集升级为V2版本，启动后会自动生成，可以增加自定义规则（原始版本转换到ruleset会不兼容）

升级功能 #

  1. 从.Net 6.0升级到.Net 8.0 ; 需要下载安装[Microsoft .NET 8.0 Desktop Runtime ]
  2. 如果遇到启动失败的情况，可以手工下载到新文件夹测试；测试时如果新的可以启动，关闭v2rayN后拷贝旧guiConfigs文件夹覆盖到新文件夹即可迁移配置
  3. 注意： 因为go v1.21以后不支持windows7 系统，所以所有最新core在windows7上可能运行失败； 解决办法是去下载对应core旧一点的版本覆盖，比如xray 的 v1.8.4

发布文件介绍 #

windows x64

  • v2rayN-With-Core.zip 包含所有Core文件，如您第一次请下载此包
  • v2rayN.zip 不包含Core文件
  • zz-v2rayN-With-Core-SelfContained.7z，包含所有Core文件；独立包，不需要下载安装Microsoft .NET 6.0 Desktop Runtime

windows x86

  • v2rayN-32.zip 不包含Core文件

windows arm64

  • v2rayN-arm64.zip 不包含Core文件
